# Blind-Man-Shoe
## 📌 Description
This project is an assistive wearable device designed for visually impaired people. It detects obstacles using ultrasonic sensors and alerts the user using vibration or buzzer.

## 🛠 Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Buzzer / Vibrator
- Battery

## ⚙️ Working
The ultrasonic sensor sends sound waves and calculates distance. If an obstacle is detected within a certain range, the buzzer alerts the user.

## 🎯 Applications
- Helps visually impaired people
- Obstacle detection system
